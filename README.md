# RTL00 Test Minimal AmebaV3.5a GCC
---
Толко Тест!<br>
Проект не завершен!<br>

В Eclipse заданы 3 опции Manage Configurations:<br>
1. AutoMake (собрать проект для прошивки, используются установки в Eclipse)<br>
2. Default (собрать проект для прошивки, используя makefile из project.realtek_ameba1_va0_example/GCC-RELEASE)<br> 
3. Realise (собрать проект, прошить используя makefiles в корне)<br>

По умолчанию, в качестве JTAG используется J-Link STLink V2.<br>
